About PyChatRoom


PyChatRoom is a cross-platform communication application specialising is text-based media conversations.

There is no development for PyChatRoom to become global, nor developed on a mass scale. It is a simple project for a greater understanding into programming.

PyChatRoom started development on 3rd April 2019 and is still currently in development. Studio Cogswork,Ltd. (A non-legimate, project company), is currently developing the project with a one man guy building the project, TheOnePath (Alias.)

Special Thanks: Whilst this is a standalone project, research on development is made possible by the Python community.
